# Dyad Briefs

This directory stores subagent evidence briefs used to rewrite the CQECMPLX
papers.

These briefs are not final papers. They are evidence maps and draft guidance.
The final review-facing papers in `Papers/Source/` are assembled separately by
the main synthesis pass.

Each brief should identify:

- scientific claim focus,
- proof and evidence files,
- verifier/receipt status,
- integer paper versus `.25/.50/.75` companion split,
- dyad cross-links,
- obligations and falsifiers,
- recommended final-paper structure.

The dyad pattern is:

```text
Block 01:  1/8,   2/7,   3/6,   4/5
Block 02:  9/16, 10/15, 11/14, 12/13
Block 03: 17/24, 18/23, 19/22, 20/21
Block 04: 25/32, 26/31, 27/30, 28/29
```
