# CQECMPLX-Repo-Forms-Accounting

Status: `trackable-index`

Route: `production/workspace-corpus-index`

## Composite Identity

`CQECMPLX-Repo-Forms-Accounting` is the literal accounting and curated repo-form
route for all known repo roots and zip archives.

## Included Families

- kernel repo forms;
- repo form index;
- per-repo `FORM.md` and `FORM.json`;
- archive exclusion files;
- literal accounting summaries;
- zip inventory metadata;
- deep-dive sanity profiles.

## Deferred Families

- raw zip archives;
- full file indexes when too large;
- SQLite accounting database;
- nested archive bodies;
- database payload bodies.

## Production Rule

Publish indexes, summaries, and curated forms before any raw archive or large
database payload. Preserve literal accounting without dedupe until a separate
dedupe/consolidation phase is declared.
