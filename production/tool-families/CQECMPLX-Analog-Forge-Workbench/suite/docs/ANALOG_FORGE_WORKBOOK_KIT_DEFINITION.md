# Analog Forge Workbook Kit - Full Definition

The Analog Forge Workbook Kit is an eightfold, color-dynamic, hand-operated reasoning system. Each color family contains eight copies of every tool so that all visible local readout states can be instantiated at once.

## Base loop

```text
observed action -> loose grey substrate -> three-color gradient -> yes/no continuation -> proof or obligation -> receipt -> bound page or open obligation
```

## Color law

Color is the only dynamic field. All tools are carriers, substrates, constraints, or records.

## Digital equivalence

Anything done physically must be recordable digitally. Anything done digitally must be reproducible physically.
