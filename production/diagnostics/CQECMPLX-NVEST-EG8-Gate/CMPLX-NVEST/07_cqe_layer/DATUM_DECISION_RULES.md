# CMPLX-NVEST Datum Decision Rules

Most NVEST outputs are library-bound diagnostics.

## Library-Bound

- wave report
- centroid residual
- estimator score
- gate result
- candidate point
- hidden-result training example
- product recommendation
- risk/value label

## Candidate New Datum

Only create a new datum when the confirmed event changes the map and cannot be represented as a receipt, morphon, diagnostic trace, adapter output, or product view.
