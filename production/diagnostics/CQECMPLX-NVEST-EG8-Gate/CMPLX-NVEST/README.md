# CMPLX-NVEST

CMPLX-NVEST is the CMPLX-side umbrella for the market/wave tool family.

NVEST means investigate and invest: not finance-only, but a general diagnostic surface for finding where a system carries value, risk, error, success, failure, phase shift, and centroid tension.

It applies the Forge-family method to any domain where a signal can be decomposed into waves, centroids, regimes, gates, and active decision points.

## Identity

```text
N = navigate / network / node
VEST = investigate, value-estimate, vector-estimate, vested identity
NVEST = investigate the applied place and bind the result to the identity of the tool and target
```

## Relationship To MORSR

CMPLX-NVEST and CQE-E/G8 together do what MORSR does in the market/wave family:

```text
Investigate / NVest
  -> E8 decomposition
  -> Gate / G8 decision
  -> hidden-result validation
  -> library-bound diagnostic map
```
