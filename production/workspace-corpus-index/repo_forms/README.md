# Curated Repo Forms

This folder is the kernel's curated view of the git-hosted roots.

It deliberately does not include zip/tar/database/archive payloads. Each repo is represented by FORM.md, FORM.json, file_index.curated.txt, and archive_exclusions.txt.

This gives the sidecar kernel a production-aware map of the repos without turning the kernel into a bulk archive carrier.
