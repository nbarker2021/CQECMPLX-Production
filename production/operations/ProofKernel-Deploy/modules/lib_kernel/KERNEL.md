# Lib Kernel

Above-kernel library memory: rules, reusable methods, proofs, and forge-library surfaces that should be pulled by identity instead of routed internally.

## Position

This companion kernel lives above and around the main `CMPLX-Kernel\kernel` sidecar. It does not replace the main kernel; it gives that kernel a structured way to find and apply the libraries, lattice readers, and Forge-family package identities without carrying bulk payloads.
